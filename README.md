## 📌 Nasıl Oynanır?

- **SPACE** veya **Tıklama**: Zıplarsın (2 kez zıplayabilirsin)
- **F** tuşu: Tam ekran moduna geç
- **M** tuşu: Müziği ve sesleri aç/kapat
- Amaç: 10 coin toplayarak kazanmak  
- Düşersen oyunu kaybedersin  
- Troll platformlara dikkat!
